# 更新日志

## v1.0.2 (2026-04-29)

### 🚀 改进

- **命令格式优化**：将 `/mslx` 后的中文子命令改为英文缩写（如 `/mslx ser restart`），方便快速输入。
- **新增 `/mslx help` 命令**：显示所有可用命令的帮助列表。

### 📝 文档

- 更新 README.md 中命令表格和使用示例为英文缩写格式。

## v1.0.1 (2026-04-29)

### 🐛 修复

- **修复认证方式**：将 `Authorization: Bearer` 改为 `x-api-key` Header，与 MSLX API 规范保持一致。

### 🔒 安全

- **新增管理员权限检查**：所有命令（服务器列表、启动/停止/重启服务器、隧道列表/详情、启动/停止/重启隧道）现在仅限管理员使用。非管理员用户执行时会收到"权限不足"的提示。

### 🚀 改进

- **命令格式统一**：所有命令前缀改为 `/mslx` 格式（如 `/mslx 重启服务器`），避免与其他插件命令冲突。

### 📝 文档

- 更新 README.md 中的配置说明，明确 `api_token` 通过 `x-api-key` Header 传递。
- 在功能表格中添加"权限"列，标明各命令的使用权限。
- 更新 README.md 中所有命令示例为 `/mslx` 前缀格式。
